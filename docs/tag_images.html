Astronomical Image Tagging/Setup/Properties


   In this window you can get informations about your image and set your image properties (tag image) with data about the author, location from where this image was shot, wheather coditions, light pollution conditions, telescope/mount and camera/settings used as well as the software used for registration, stacking and processing. Additionaly you can add/records various notes to the image.
