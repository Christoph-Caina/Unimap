Astronomical Distances Measurement Tool


   If astrometry data (parallax, red shift) is available for the sky objects in the image, there is a tool available to measure various distances in the sky:

  • Distance in image pixels between the selected points
  • Angular distance in the sky - in degrees
  • the exact distance between two stars - if data available both stars
  • Average galactic distance - calculated based on average background stars
  • Distance between galaxies - if data available
  • Average extra-galactic distance - calculated over background galaxies in the area
