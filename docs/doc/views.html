Unimap Interface - Views


    There are several views/modes you should be aware of:

  • Main image view - where the image is loaded and where you can interact with your image

  • Video camera view will be rendered in the same main view - only the user interaction will be slightly different

  • Group thumbnails view - used to display icons for all images in the group and which can also be used to select you current image

  • Main status bar where you can follow zoom levels, current position in the image and in sky and number of detected or matched objects
