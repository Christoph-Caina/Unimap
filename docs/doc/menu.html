Unimap Interface - Menu


    The main menu has the following sections:

  • File - handles file input out operations
  • View - to customize the main views
  • Group - operations on groups/sequences of images
  • Image - processes on astro- image data
  • Tools - a collection of astronomy related modules
  • Options - here you can setup all application parameters
  • Help - calls this manual as well as a feedback interface
