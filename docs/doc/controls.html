Unimap Interface - Controls


    Mouse and keyboard controls in image view:

  • Zoom in/out : position you mouse pointer on the point you whish to zoom and action the mouse wheel.

  • Dragging the image: keep the mouse's left click pressed and by moving the mouse you can drag the image around.

  • With the mouse's left double click you can call the object details dialog or in instruments view with guiding/tracking you can activate the guide/track point.

  • On right click a menu will pop up from where you can add a note on the spot, action on areas, view current position in the sky with external applications or slew you telescope to that position.

  • When using the zoom tool you use Alt key to zoom out and you can use the Ctrl key to drag the image around.
