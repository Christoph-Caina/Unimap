Unimap Images - Processing

    Several processing filters have been implemented so far:

  • Operators: Negative, Addition, Subtraction and Multiplication
  • Colors: Brightness/Contrast/Gamma, Midtone, Balance, Hue/Saturation
  • Filters: Blur, Sharp, Deconvolution, Gradient, Morphological, Digital
  • Astro: Denoising, Debloomer

    Additionally you can export the image to your preferred editing application (Photoshop, Gimp, etc) by Menu->Image->Process With...
