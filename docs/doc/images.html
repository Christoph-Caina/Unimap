Unimap - Images Tasks


    There are various tasks the application can do with images and sequences:

  • Image tagging - setting up the date/time, site, target, weather conditions, software used, etc

  • Detection and matching of the image against the star catalogs

  • Processing image using several filters

  • Call to external application for processing such as: Adobe Photoshop, Gimp, etc.

  • Registering and stacking of a defined sequence of images

  • Automatic discovery of objects and paths
